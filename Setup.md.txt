# sort-orders
Teste de ordenação de pedidos

Para a melhor utlização e execução deste projeto é importante que você tenha instalado em sua máquina: Visual Studio 2017 ou posterior / Visual Studio Code.

# Como baixar esse projeto:
 - Faça o fork deste do repositório;
 - Com o fork do repositório realizado, você está pronto para cloná-lo e ter uma cópia local
 - Clone o Repositório.

# Como executar 
 - Abra o projeto no visual estudio;
 - Compile o projeto e execute

# Como utilizar o sistema
Ao executar o projeto, será exibida a tela inicial com um menu. Navegue por ele através das opções disponibilizadas.
- O menu apresenta de forma interativa as funcionalidades existentes e validações solicitadas

# Entendendo o Menu

# Opção 1:Adicionar item ao carrinho 

Solicita o nome e o valor do produto, após informá-los basta pressionar
Enter para adicioná-lo.

# Opção 2: Um unico produto no carrinho
Solicita o nome que deve ser buscado, caso exista o retorno será o nome 
deste mesmo produto informado. Enter para retornar ao menu.

# Opção 3: Carrinho não ordenado 
Mostra os produtos no carrinho ordenados em ordem crescente
Enter para retornar ao menu.

# Opção 4:Carrinho ordenado  em ordem crescente
Mostra os produtos no carrinho ordenados em ordem crescente
Enter para retornar ao menu.

# Opção 5:Carrinho ordenado  em ordem decrescente
Mostra os produtos no carrinho ordenados em ordem decrescente
Enter para retornar ao menu.

# Opção 6: Produtos de maior e menor valor
Mostra o produto com o maior valor e o produto com menor valor.
Enter para retornar ao menu.

# Opção 0: Sair
Ao escolher esta opção pressione o Enter 2 vezes para sair
